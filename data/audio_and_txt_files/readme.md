# Respiratory Sound Classification

## 📌 Project Overview

This project focuses on classifying respiratory (lung) sounds using machine learning and deep learning techniques. The goal is to detect abnormalities such as wheezes and crackles from audio recordings.

---

## 📂 Dataset

We use the **ICBHI Respiratory Sound Dataset** available on Kaggle.

🔗 Download dataset:

* https://www.kaggle.com/datasets/vbookshelf/respiratory-sound-database

### 📊 Dataset Details

* 920+ respiratory audio recordings
* Annotation (.txt) files
* Patient diagnosis information
* Multiple respiratory conditions (COPD, pneumonia, etc.)

---

## ⚠️ Important Note (Dataset Setup)

Due to GitHub file size limits, the dataset is NOT included in this repository.

### 👉 Steps to use dataset:

1. Download dataset from Kaggle
2. Extract the dataset
3. Replace the folder:

```bash
data/audio_and_txt_files/
```

with your downloaded dataset folder.

### Final structure should look like:

```
 ├── data/
      ├── audio_and_txt_files/   ← replace with downloaded data
```
